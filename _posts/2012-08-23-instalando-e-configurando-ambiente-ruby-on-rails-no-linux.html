Fala pessoal, beleza? Alguém utiliza Ruby on Rails</strong> ai? Sabe o que é? Se você não sabe o que é não deixe de ver esse post</a> onde explico um pouco sobre essa incrível ferramenta.

Nesse post irei explicar como instalar e configurar um ambiente de desenvolvimento Ruby on Rails em Linux, em meu caso estou usando a distribuição Ubuntu 12.04 LTS.

</a>

Importante (atualizado 13/07/2013):</strong> esse tutorial demonstra a instalação do Ruby 1.9.3, recentemente saiu a versão 2.0.0 da linguagem. É recomendável o uso dessa nova versão com o também recém lançado Rails 4. Após seguir os passos abaixo, leia esse post onde explico como atualizar.</a></div>

      Atualizando lista de pacotes</h2>
      Após estar com nosso sistema instalado, vamos atualizar a lista de pacotes, para isso execute no terminal o seguinte comando:
      [code language="bash"]sudo apt-get update[/code]

      Gerenciador de Versões do Ruby</h2>
      Primeiro de tudo, precisamos instalar a chave pública do autor do RVM (Ruby Version Manager):
      [code language="bash"]gpg --keyserver hkp://keys.gnupg.net --recv-keys D39DC0E3[/code]

      Vamos instalar o RVM para caso precisarmos ter outras versões do Ruby instalado na máquina conseguirmos facilmente, rode o comando curl -L https://get.rvm.io | bash -s stable</code> como abaixo.</div>
      </a>

      Você precisa ter a biblioteca cURL instalada para executar o comando acima, caso não tenha, basta rodar o comando: sudo apt-get install curl</code>.</div>

      Se tudo ocorreu bem agora você deve ter o RVM instalado, vamos adicionar ele ao shell, execute o comando abaixo.</a></p>
      Se tudo ocorreu bem agora você deve ter o RVM instalado, vamos adicionar ele ao shell, para isso adicione o código abaixo no arquivo .bashrc</code> ou .bash_profile</code> na home do usuário:
      [code language="bash"]PATH=$PATH:$HOME/.rvm/bin # Add RVM to PATH for scripting
      [[ -s "/usr/local/rvm/scripts/rvm" ]] && source "/usr/local/rvm/scripts/rvm"
      [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm"[/code]
      O código acima serve para toda vez que abrirmos o terminal podermos executar comandos do RVM apenas usando rvm</code>.

      Caso algum comando com rvm</code> não funcione daqui pra frente, você deve acessar as preferências (Edit -> Profile Preferences</code>) do terminal e marcar a opção para executar comandos com login.</div>
      </a>

      Instalando o Ruby</h2>
      Agora temos que instalar a versão do Ruby que desejamos usar, antes, vamos ver as versões disponíveis usando o comando rvm list known</code>.       </a>

      O Ruby possui vários interpretadores (JRuby, Rubinius, MacRuby entre outras) mas nós vamos instalar a mais comum, que é a MRI (Matz's Ruby Interpreter), a versão implementada pelo criador da linguagem Ruby, o japonês Yukihiro Matsumoto ("Matz").

      Antes de instalarmos o Ruby, vamos instalar algumas dependências necessárias, para isso execute o comando sudo apt-get install build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev ncurses-dev automake libtool bison subversion pkg-config</code>. </a>

      Observe que estamos instalando o sistema de versionamento Git também através do pacote git-core</code>.</div>
      Vamos ao Ruby agora, rode o comando rvm install 1.9.3</code> também para instalarmos a última versão do Ruby MRI (como vista pelo comando rvm list known</code> acima).
      </a>

      Como o RVM suporta várias versões do Ruby (pra isso que ele foi criado), devemos escolher uma versão padrão que usaremos, para isso, execute o comando rvm use 1.9.3 --default</code> após a instalação do Ruby 1.9.3.</div>
      </a>

      Já temos o Ruby instalado, podemos executar o comando ruby -v</code> no terminal para verificarmos a versão, veja acima.

      O Ruby também possui um IRB (shell interativo ruby em português) onde podemos executar código diretamente, para acessar, basta digitar irb</code> no terminal. Abaixo mostro como fazer um simples soma.</a>

      Digite quit</code> para sair do IRB</div>

      Instalando Ruby on Rails</h2>
      Agora vamos à instalação do Ruby on Rails (finalmente), execute o comando gem install rails</code>.       </a>

      Aguarde um tempo e já teremos o Rails instalado, após o término, rode o comando rails -v</code> para verificar a versão.
      </a>

      Instalando MySQL</h2>
      Já temos o Ruby e o Ruby on Rails instalados, se você quiser já pode iniciar um projeto Rails usando o banco de dados SQLite, mas como eu quero usar MySQL, irei instalá-lo através do comando sudo apt-get install mysql-server</code>, veja abaixo.       </a>

      Você precisará instalar a biblioteca libmysqlclient-dev</code> também via sudo apt-get install libmysqlclient-dev</code></div>

      Durante a instalação do MySQL ele irá pedir a senha do usuário root, esse usuário é o administrador, é recomendado definir uma, caso não queira, basta apertar enter para deixar vazia.

      Instalando PostgreSQL</h2>
      Caso você precise do PostgreSQL, então vamos ao passo-a-passo.

      Primeiro verifique qual a última versão disponível no repositório do Ubuntu, rode apt-cache search postgresql</code> e na letra p</code> veremos o último release. Agora instale o PostgreSQL através de sudo apt-get install postgresql-9.1</code> (substitua 9.1 pela versão desejada) e depois instale uma biblioteca necessária</u> para trabalhar com Ruby: sudo apt-get install libpq-dev</code>.

      Agora precisamos alterar a senha, para isso rode sudo -u postgres psql template1</code> no terminal, então irá aparecer template1=#</code> na tela, digite \password postgres</code> e então digite e confirme a nova senha e aperte Ctrl+d</code> para sair. Agora reinicie o serviço usando sudo /etc/init.d/postgresql restart</code>.

      Instalando PgAdmin3</h3>
      Se você deseja ter uma interface mais amigável para trabalhar com o banco pode instalar o PgAdmin3, apenas execute sudo apt-get install pgadmin3</code> e pronto!

      Instalando Aptana Studio</h2>

      Atualizado 13/07/2013:</strong> faz algum tempo que migrei do Aptana Studio para o Sublime Text</a>, recomendo!</div>
      Com o banco de dados instalado, agora estamos quase acabando, só falta instalar a IDE (ambiente de desenvolvimento integrado em português), eu particularmente gosto do Aptana Studio que é baseado no Eclipse, acho muito versátil, uso ele para trabalhar com PHP, Python, Ruby, Javascript, HTML e CSS. Caso queira instalar ele, basta fazer o download no site oficial       aqui</a>.

      Após o download, extraia o arquivo na pasta preferida, porém ele ainda não irá funcionar porque como é feito em Java, precisamos instalar a máquina virtual Java (JRE), para isso, basta executar o comando sudo apt-get install openjdk-6-jre</code> no terminal e pronto, basta clicar no ícone AptanaStudio3</code> e teremos nosso ambiente de desenvolvimento.</a>

      É isso ai pessoal, o que vocês acharam? Muito difícil?

      Abraços.
