Fala pessoal, beleza?

Estou aqui para falar de Linux hoje. Em meu computador tenho uma partição NTFS</strong> e sempre quando vou acessar algum arquivo pelo Ubuntu</strong> 12.04 preciso esperar o sistema montar ela. A pesquisa do dash também não funciona pois o sistema não reconhece as partições ainda não montadas.

Para resolver isso, tudo que precisamos fazer é executar um comando ao iniciar do Ubuntu que faça a montagem automaticamente para nós.

Adicione o comando abaixo na inicialização (clique aqui</a> caso não saiba) trocando o sda2</code> pela partição a ser montada:
[code language="bash"]udisks --mount /dev/sda2[/code]
Lembrando que o código acima deve ser colocado no campo Command</code>, feito isso salve e pronto!

Se você conhece apenas o rótulo da partição, como ARQUIVOS</code> ou Disco C</code> por exemplo, rode o comando abaixo no terminal:
[code language="bash"]sudo blkid[/code]
Ele irá exibir uma saída como essa onde facilmente podemos ver no canto esquerdo a partição que desejamos.
