Fala pessoal! Segue abaixo uma lista com os principais comandos de git usados no dia-a-dia, com o passar do tempo vou adicionando mais..

Geral

Adicionar todos arquivos alterados no stage (deixar pronto para serem comitados):

{% highlight bash %}git add .{% endhighlight %}

Adicionar arquivos trackeados excluídos no stage:

{% highlight bash %}git add -u{% endhighlight %}

Remover arquivos do stage (prontos para commit) que já são trackeados:

{% highlight bash %}git reset HEAD nome_arquivo{% endhighlight %}

Remover arquivos do stage que ainda não são trackeados:

{% highlight bash %}git rm --cached nome_arquivo{% endhighlight %}

Refazer um commit quando necessário:

{% highlight bash %}git commit -m "mensagem" --amend{% endhighlight %}

Exibir alterações feitas depois do último commit que ainda não foram stageadas:

{% highlight bash %}git diff{% endhighlight %}

Exibir alterações feitas depois do último commit que já foram stageadas:

{% highlight bash %}git diff --cached{% endhighlight %}

Excluir um commit local (substitua o número abaixo pelo seu commit)

{% highlight bash %}git reset --hard 7d01b96b7fd1a1e5bfa1971b50c51e55efa75646{% endhighlight %}

Forçar o git a excluir um commit no repositório remoto, primeiro exclua localmente e depois rode:

{% highlight bash %}git push origin HEAD --force{% endhighlight %}

Abrir interface gráfica (gitk):

{% highlight bash %}gitk{% endhighlight %}

Ver número de commits por usuário:

{% highlight bash %}git shortlog -s{% endhighlight %}

Stash

Jogar alterações para o stash:

{% highlight bash %}git stash{% endhighlight %}

Ver itens no stash:

{% highlight bash %}git stash list{% endhighlight %}

Aplicar o último item adicionado no stash:

{% highlight bash %}git stash apply{% endhighlight %}

Aplicar um stash específico (liste todos os stashs e substitua 'numero_do_stash' pelo número desejado):

{% highlight bash %}git stash apply stash@{numero_do_stash}{% endhighlight %}

Limpar stash:

{% highlight bash %}git stash clear{% endhighlight %}

Branches

Criar um branch:

{% highlight bash %}git branch nome_branch{% endhighlight %}

Criar um branch e dar checkout (mudar pra ele):

{% highlight bash %}git checkout -b nome_branch{% endhighlight %}

Criar um branch "trackeando" (fazendo acompanhar) um branch remoto:

{% highlight bash %}git checkout -b nome_branch_local origin/nome_branch_remoto{% endhighlight %}

Mudar de branch:

{% highlight bash %}git checkout nome_branch{% endhighlight %}

Excluir um branch local:

{% highlight bash %}git branch -d nome_branch{% endhighlight %}

Excluir um branch remoto:

{% highlight bash %}git push origin --delete nome_branch{% endhighlight %}

Exibir todos os branches locais:

{% highlight bash %}git branch{% endhighlight %}

Exibir todos os branches locais e remotos:

{% highlight bash %}git branch -a {% endhighlight %}

Fazer o git enxergar todos branches remotos:

{% highlight bash %}git remote update{% endhighlight %}

Ver origens remotas definadas para o branch atual:

{% highlight bash %}git remote show {% endhighlight %}

Excluir origem remota para o branch atual:

{% highlight bash %}git remote rm nome-origin{% endhighlight %}

Adicionar origem remota para o branch atual:

{% highlight bash %}git remote add nome-origin git@xxxbucket.com:username/nome-repositorio.git{% endhighlight %}

Trazer ("mergear") alterações de outro branch para o branch atual:

{% highlight bash %}git merge nome-branch-a-ser-baixado {% endhighlight %}

Ver apontamentos de branches do git, na pasta do .git, rode:

{% highlight bash %}cat .git/config {% endhighlight %}

Configuração

Definir configuração de nome global:

{% highlight bash %}git config --global user.name "Seu nome" {% endhighlight %}

Definir configuração de email global:

{% highlight bash %}git config --global user.email "email@dominio.com.br"{% endhighlight %}

Colorizar saídas:

{% highlight bash %}git config --global color.ui auto {% endhighlight %}

Criar alias (cria um alias "st" para "status", agora pode-se usar git st):

{% highlight bash %}git config --global alias.st 'status' {% endhighlight %}

Definir editor padrão do git (utilizado em merges etc)

{% highlight bash %}git config --global core.editor "vim"{% endhighlight %}
