Está tendo problemas com a função scroll</a> do jQuery</strong> no Internet Explorer</strong>?

Se você estiver usando document</code> como seletor, não irá funcionar no IE, veja no exemplo abaixo:
[code language="javascript"]// Não funciona no IE
$(document).scroll(function(){

// seu código aqui

});[/code]
ou
[code language="javascript"]// Não funciona no IE
$(document).on('scroll', function(){

// seu código aqui

});[/code]
A solução cross browser para isso é utilizar window</code> como seletor:
[code language="javascript"]// Cross browser
$(window).scroll(function(){

// seu código aqui

});[/code]
ou
[code language="javascript"]// Cross browser
$(window).on('scroll', function(){

// seu código aqui

});[/code]
