Está tendo problemas com a função scroll do jQuery no Internet Explorer?

Se você estiver usando document como seletor, não irá funcionar no IE, veja no exemplo abaixo:

{% highlight javascript %} // Não funciona no IE $(document).scroll(function(){ // seu código aqui }); {% endhighlight %}

ou

{% highlight javascript %} // Não funciona no IE $(document).on('scroll', function(){ // seu código aqui }); {% endhighlight %}

A solução cross browser para isso é utilizar window como seletor:

{% highlight javascript %} // Cross browser $(window).scroll(function(){ // seu código aqui }); {% endhighlight %}

ou

{% highlight javascript %} // Cross browser $(window).on('scroll', function(){ // seu código aqui }); {% endhighlight %}
