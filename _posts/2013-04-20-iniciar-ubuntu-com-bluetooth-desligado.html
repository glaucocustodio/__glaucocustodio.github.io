Hoje venho mostrar a vocês como configurar o Ubuntu</strong> para iniciar com o bluetooth desligado</strong>. Essa é uma boa dica para quem usa notebook e precisa economizar bateria.

Em meu caso estou usando a versão 12.04 LTS do sistema, mas creio que essa solução também sirva para 12.10 e similares..

Abra seu terminal e digite o seguinte código para editarmos o arquivo rc.local</code>: sudo gedit /etc/rc.local</code>

</a>

Digite a senha do usuário e quando o editor se abrir, adicione rfkill block bluetooth</code> antes de exit 0</code>.

</a>

Salve o arquivo e pronto, toda vez que o sistema iniciar o bluetooth estará desligado por padrão, caso precise, você pode ligá-lo manualmente perto do relógio.

</a>

Essa foi a dica de hoje.
