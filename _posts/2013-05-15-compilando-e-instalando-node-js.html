Fala pessoal, hoje venho com meu primeiro post sobre Node.js</strong>.

Pra quem não conhece, essa é uma plataforma feita sobre a engine V8 do navegador Chrome que possibilita a construção de aplicações escaláveis que funcionam muito bem em tempo real (real-time).

</a>

Meu objetivo aqui é mostrar como baixar, compilar e instalar</strong> essa ferramenta no Linux, Ubuntu 12.10 pra ser mais preciso, mas creio que deva funcionar para a maioria das outras distribuições. Se você é desenvolvedor e está no Windows, sugiro que use um sistema *nix like pra isso ao invés de tentar outras soluções alternativas.

Muitos caminhos, qual o melhor?</h2>
Por ai a fora na internet encontramos muitas formas de instalar o Node, dentre algumas que vi estão:

  • Node Version Manager - NVM</a></li>
  • nave</a></li>
  • Usar package manager do sistema operacional</a></li>
  • Baixar uma versão específica via git ou não, compilar e instalar</li>
    </ul>
    Eu particularmente gostei da forma que clonamos o repositório via git, compilamos e instalamos (essa que veremos aqui). Me parece ser simples, rápida (não precisa mecher com arquivos de perfil manualmente) e fácil de atualizar.

    Qual a melhor forma? Não sei, é uma questão de gosto, fica a seu critério. Se você faz de uma forma diferente escreva nos comentários abaixo..

    Pré-requisitos</h2>
    Antes de instalarmos o Node, precisamos satisfazer os pré-requisitos necessários..

    Comece atualizando a lista de pacotes:
    [code language="bash"]sudo apt-get update[/code]
    Instale o compilador:
    [code language="bash"]sudo apt-get install build-essential[/code]
    Como vamos clonar o repositório precisamos do git instalado, instale-o caso ainda não tenha:
    [code language="bash"]sudo apt-get install git-core[/code]
    Precisamos do Python (versão 2.6 ou 2.7), mas isso o Linux já tem por padrão.. Caso precise rode:
    [code language="bash"]sudo apt-get install python[/code]
    Instale demais bibliotecas dependentes:
    [code language="bash"]sudo apt-get install g++ curl libssl-dev apache2-utils[/code]

    Compilando e Instalando</h2>
    Agora que temos o necessário, vamos instalar o Node.

    Acesse a pasta que deseja instalar, clone o repositório do Node (pode demorar um pouco) e entre no diretório criado:
    [code language="bash"]cd /usr/bin[/code]
    [code language="bash"]sudo git clone git://github.com/joyent/node.git[/code]
    [code language="bash"]cd node[/code]
    Acesse o     site do Node</a>, veja a última versão estável e mude para o branch dela:
    [code language="bash"]sudo git checkout v0.10.6-release[/code]
    Configure o código para ser compilado, compile e instale:
    [code language="bash"]sudo ./configure[/code]
    [code language="bash"]sudo make[/code]
    [code language="bash"]sudo make install[/code]
    Agora já temos o Node.js instalado, rode node -v</code> para ver a versão:

    </a>

    Rodando..</h2>
    Finalizando, vamos fazer um teste para confirmar se está tudo certo.

    Crie um arquivo chamado first.js</code> e adicione o código abaixo:
    [code language="javascript"]var http = require('http');

    http.createServer(function(request, response){
    response.writeHead(200);
    response.end("Node.js rodando..");
    }).listen(8080);

    console.log('Rodando na porta 8080');[/code]
    No diretório do arquivo criado, rode node first.js</code> no terminal. Agora abra o navegador no endereço http://localhost:8080/</code> e você deverá ver a seguinte mensagem:

    </a>
