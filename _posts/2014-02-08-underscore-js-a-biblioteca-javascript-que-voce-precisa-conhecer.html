Hoje estou aqui para compartilhar uma biblioteca JavaScript que conheci a pouco tempo e que você também precisa saber de sua existência!

Acredito que os programadores Ruby estão acostumados com métodos simples como: each, map, min, max</code> etc.. Agora você pode ter eles em JavaScript, com a biblioteca Underscore.js</a>.

Escolhi alguns métodos para apresentá-los, vejamos abaixo..

Each</h2>

Itera sobre uma lista de elementos, podendo acessar o valor e índice.</p>
[code language="javascript"]
_.each([10, 20, 30, 40], function(v, i){
// listando [valor, índice]
console.log([v, i]);
});
=> [10, 0]
=> [20, 1]
=> [30, 2]
=> [40, 3][/code]

Map</h2>

Produz um novo array mapeando cada valor através da função.</p>
[code language="javascript"]
_.map([2, 4, 6],function(v){
// multiplica por 2
return v * 2;
});
=> [4, 8, 12]
[/code]

Filter</h2>

Percorre cada elemento, retornando um array de valores que passam na condição.</p>
[code language="javascript"]
_.filter(['João', 'Pedro', 'Paulo'], function(v){
// apenas que começam com a letra "P"
return v.charAt(0) == 'P';
});
=> ["Pedro", "Paulo"]
[/code]

Contains</h2>

Retorna true</code> se o valor está presente na lista.</p>
[code language="javascript"]
_.contains(['ruby', 'python', 'php'], 'javascript');
=> false
[/code]

Max</h2>

Retorna o valor máximo da lista.</p>
[code language="javascript"]
_.max([23, 45, 65, 34, 1, 35])
=> 65
[/code]

Shuffle</h2>

Retorna uma cópia misturada da lista.</p>
[code language="javascript"]
_.shuffle([1, 2, 3, 4, 5]);
=> [5, 2, 1, 4, 3]
[/code]

Last</h2>

Retorna o último elemento do array.</p>
[code language="javascript"]
_.last([{name: 'André', age: 21}, {name: 'Marta', age: 37}]);
=> Object { name="Marta", age=37}
[/code]

Flatten</h2>

"Achata" um array.</p>
[code language="javascript"]
_.flatten([[1, 2], [[[4, 5]]], [[[[6], 7]]]])
=> [1, 2, 4, 5, 6, 7]
[/code]

Range</h2>

Cria um faixa de valores.</p>
[code language="javascript"]
// Começa em 100, vai até 200, de 10 em 10
_.range(100, 210, 10);
=> [100, 110, 120, 130, 140, 150, 160, 170, 180, 190, 200]
[/code]

Has</h2>

Retorna true</code> se o objeto contém uma dada chave.</p>
[code language="javascript"]
_.has({a: 1, b: 2, c: 3}, 'd');
=> false
[/code]

Times</h2>

Invoca uma função iteradora dado o número de vezes.</p>
[code language="javascript"]
_(4).times(function(n){
console.log("número " + n)
});
=> número 0
=> número 1
=> número 2
=> número 3
[/code]

Essa são apenas algumas funções, entre no site</a> e confira todas.
